펜타캠퍼스 사이트 기획 요청서

펜타랩이 펜타캠퍼스 웹사이트 구축을 시작하기 전, 발주처의 의도·타겟·범위를 정확히 수집하기 위한 문서입니다. 답변 내용을 기준으로 기획서(PRD)·견적서·작업범위서가 작성됩니다.

안녕하세요, 대표님.

펜타캠퍼스(연수원) 사이트 구축을 본격 진행하기 전에, 대표님의 생각을 정확히 전달받고 시작하고 싶습니다. 아래 6개 블록 · 총 20여 문항으로 구성되어 있으며, 각 질문 아래 예시 답변을 몇 가지씩 참고용으로 적어두었습니다.

예시는 그저 방향을 보여드리기 위함이니 얽매이지 마시고, 생각나는 대로 자유롭게 답변해주시면 됩니다. 한 번에 다 완성하실 필요 없고, 블록 단위로 나눠 주셔도 좋습니다.

ⓘ 답변이 회수되면 펜타랩은 이를 바탕으로 ① 기획서(PRD) ② 견적서 ③ 작업범위서를 3영업일 내 작성하여 회신드립니다.
BLOCK 01

비전

왜 이 연수원을 만들려 하시는지, 장기적으로 어디로 가고 싶으신지 여쭙는 블록입니다.

1-1
왜 연수원을 만들고 싶으세요?
예시 답변
  • 펜타스쿨에 공급할 강사 풀을 스스로 만들기 위해
  • 1인 콘텐츠 의존에서 벗어나 시스템으로 굴러가게 하려고
  • 수익 다각화 — 학원비 외 연수료 매출 라인 추가
  • 후배 교사들에게 실전 문해력 교육 노하우를 정식으로 전하고 싶어서
1-2
10년 뒤 펜타캠퍼스가 어떤 모습이면 성공이라고 느끼실까요?
예시 답변
  • 누적 수료생 1만 명, 전국 200개 가맹학원에 수료생이 들어가 있는 모습
  • "문해력 강사 자격증은 펜타"라는 인식이 업계 표준이 된 모습
  • 교육청 위탁연수까지 운영하는 종합 교원연수원으로 확장
1-3
만약 6개월 안에 연수원이 없어진다면 가장 아쉬운 것은 무엇일까요?
예시 답변
  • 예비 강사들을 배출할 통로가 사라지는 것
  • 펜타스쿨 가맹 확장이 1인 강사 의존에서 못 벗어나는 것
  • 교재로만 전달하던 교육관을 강의로 전할 기회 상실
BLOCK 02

타겟 수강생

이 연수를 듣게 될 사람은 누구인지, 그들이 어떤 상태에 있는지 파악합니다.

2-1
수강생은 누구인가요? (1순위 / 2순위 / 3순위 나눠서)
예시 답변
  • 1순위: 30~40대 경력단절 여성, 교직·독서지도 경험자 / 2순위: 현직 초등교사 / 3순위: 문해력 학원 가맹 희망 원장
  • 1순위: 엄마표 교육에 익숙한 학부모 / 2순위: 대학원 교육학과 재학생 / 3순위: 공교육 은퇴 교사
  • 1순위: 펜타스쿨 가맹 예정자 / 2순위: 독서논술 학원 강사 / 3순위: 방과후 강사 지원자
2-2
그분들이 지금 겪고 있는 문제·고민은 무엇일까요?
예시 답변
  • 자격증을 따도 정작 취업할 곳이 없다
  • 아이스크림·티처빌 연수는 많지만 '문해력 지도'를 실전까지 가르쳐주는 곳은 없다
  • 공부한 내용을 내 아이에게만 쓰고 끝나는 게 아쉽다
  • 학원 차리고 싶은데 커리큘럼·노하우가 없다
2-3
그분들이 지금은 어디서 연수를 듣고 있나요? (대체재)
예시 답변
  • 아이스크림 원격교육연수원 · 티처빌
  • 교대·사대 부설 평생교육원
  • 유튜브 무료 강의 + 단행본 독학
  • 슬기로운초등생활 유튜브 구독, 책 구입으로 대체
BLOCK 03

5개 자격증 과정

사이트의 핵심 상품입니다. 이 블록의 답이 확정되어야 과정 상세 페이지·카드·가격이 만들어집니다.

3-1
5개 과정을 생각나시는 대로 알려주세요. (과정명 / 주제 / 담당 강사)
예시 답변 (형식 참고용)
  • ① 초등 문해력 지도사 / 초등 독해 전반 / 이은경 대표
  • ② 논증과추론 지도사 / 비판적 사고 · 논술 / 김문영 강사
  • ③ 인문독서 지도사 / 고전 · 인문 독서 지도 / 미정
  • ④ 어휘·개념어 지도사 / 사고도구어 체계화 / 미정
  • ⑤ 하브루타 토론 지도사 / 질문·토론 수업 설계 / 미정
3-2
왜 이 5개인가요? (다른 주제들 제쳐두고 이것을 고르신 이유)
예시 답변
  • 펜타스쿨 기본 커리큘럼을 가르칠 수 있는 최소 단위라서
  • 내가 직접 15년간 검증해본 과목들이라서
  • 시장에서 강사 수요가 확실한 영역이라서
  • PENTA 5영역 브랜드와 맞추기 위해서
3-3
PENTA 5영역(독해·어휘·사고·표현·소통)과 5과정은 어떻게 연결되나요?
예시 답변 (택 1)
  • ① 5영역 = 5과정 1:1 매핑 (독해지도사 / 어휘지도사 / 사고지도사 / 표현지도사 / 소통지도사)
  • ② 영역과 무관한 별도 커리큘럼 (강사진·주제 중심)
  • ③ 하이브리드 — 2~3개는 영역 기반, 나머지는 심화 주제
3-4
수강료는 얼마 정도 생각하세요? (과정당 / 5과정 묶음)
예시 답변
  • 과정당 30만 원 / 5과정 묶음 120만 원 (진입장벽 낮춤)
  • 과정당 50만 원 / 5과정 묶음 200만 원 (중가 포지션)
  • 과정당 80만 원 / 5과정 묶음 300만 원 + 가맹 우선권 (프리미엄)
BLOCK 04

성공 기준

"잘 되고 있다"를 무엇으로 판단하실지 구체적 숫자·상태로 정의해주세요.

4-1
오픈 3·6·12개월 후, 수강생 몇 명이면 성공이라 느끼실까요?
예시 답변
  • 3개월 30명 / 6개월 100명 / 1년 300명
  • 3개월 10명 / 6개월 50명 / 1년 200명 (천천히 검증)
  • 3개월 100명 / 6개월 500명 / 1년 2,000명 (공격적 목표)
4-2
오픈 1년 뒤 매출이 얼마면 만족스러우실까요?
예시 답변
  • 연 1억 원 (월 평균 800만)
  • 연 3억 원 (수강생 평균 150명 × 200만)
  • 매출은 부차적, 수료생 품질이 더 중요
4-3
수료생 중 몇 명이 실제 취업하면 성공이라 부를 수 있을까요?
예시 답변
  • 1년 내 10명 이상이 펜타스쿨·가맹점에 들어가면 성공
  • 수료생의 30% 이상이 어떤 형태로든 현장에서 활동
  • 취업은 후순위, 자기계발 완료도 의미 있음
4-4
숫자 말고, 개인적으로 "이 정도면 만족"이라고 느낄 모습은?
예시 답변
  • 월 1회 라이브 특강이 매번 꽉 찬다
  • 수료생이 자기 아이 변화 사례를 후기로 써준다
  • 강사 섭외 문의가 학교·학원에서 먼저 들어온다
BLOCK 05

하지 않을 것 (Non-goals)

경계를 명확히 하기 위한 블록입니다. 이게 확정되어야 사이트가 무한히 커지지 않습니다.

5-1
절대 하고 싶지 않은 것은 무엇인가요?
예시 답변
  • 교육청·학교 단체 위탁연수 (영업 복잡, 대금 늦음)
  • 학점은행제 학위 과정 (법규 부담)
  • 오프라인 집체 연수 (물리적 운영 부담)
  • 내 얼굴이 전면에 나오는 개인 브랜드화 (피로)
5-2
아이스크림 연수원에서 하는 것 중 펜타는 빼고 싶은 것이 있으세요?
예시 답변 (복수 선택 가능)
  • 연수 회원권(구독제) · 포인트·쿠폰 시스템
  • 교재 판매 · 교구 패키지
  • 쌤클래스(라이브+콘텐츠+소셜) 복잡 구조
  • 쇼츠연수(짧은 영상 연수)
  • 학교 위탁연수 · 단체 예산 견적
5-3
초기 3년간은 절대 손대지 않을 것은?
예시 답변
  • 해외 진출 · 영어 버전
  • 중·고등 교사 대상 확장
  • 학부모 대상 상품
  • 오프라인 스쿨 운영
BLOCK 06

현실 제약

예산·시간·법률 등 이번 프로젝트에 걸려 있는 현실 조건을 확인합니다.

6-1
예산 제약이 있나요? (마케팅·개발 투자 가능 범위)
예시 답변
  • 초기 투자 1,000만 원 이내 (첫 매출로 순환)
  • 3,000만 원까지 가능, 그 이상은 대출·투자 필요
  • 광고비는 월 100만 원 내외로만
6-2
대표님이 주당 몇 시간까지 펜타캠퍼스에 쓸 수 있으세요?
예시 답변
  • 주당 10시간 이내 (타 업무 병행)
  • 주당 20시간 (오전 전담)
  • 콘텐츠 제작 집중 기간엔 주 30시간까지 가능
6-3
지금 걸림돌이라고 느끼는 건 뭔가요?
예시 답변
  • 평생교육원 인가 서류 진행 지연
  • 강사 섭외가 제일 막막함
  • 콘텐츠 제작 시간이 부족
  • 운영·결제·CS 시스템 전무
6-4
개인 생활에서 꼭 지키고 싶은 선이 있으세요?
예시 답변
  • 주말은 절대 업무 금지
  • 저녁 9시 이후 카톡·메일 확인 안 함
  • 가족 여행·자녀 돌봄이 1순위

답변 이후 진행 절차

본 요청서 답변 회수 시점부터 펜타랩은 아래 4단계 산출물을 순차 제작해 회신드립니다. 각 단계 완료 시 대표님 승인을 받아 다음 단계로 진행합니다.

STEP 1
기획서 (PRD)
STEP 2
견적서 · 작업범위서
STEP 3
사이트 구조 · 화면 설계
STEP 4
디자인 · 개발 · 오픈
